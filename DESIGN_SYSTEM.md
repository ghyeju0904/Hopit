# 🎨 HOP - 픽셀 아트 디자인 시스템

## 📌 브랜드 개요

**HOP** = Habit + Rabbit 🐰

- **로고**: h(토끼 귀) o(트랙 링) p(점프 모션) i(당근)
- **스타일**: 16-bit 픽셀 아트 + 자연친화 색감
- **톤**: 게임 같으면서 따뜻하고 감성적인 습관앱
- **타겟**: 게이밍 감성 × 습관 추적 × 동기부여

## ✍️ 타이포그래피

### 폰트 선택
- **Display (제목)**: 픽셀 폰트 또는 Sans-serif Bold
  - 크기: 18-20px
  - 예: "오늘의 Hop 완료!"
  
- **Heading (소제목)**: Sans-serif SemiBold
  - 크기: 14-16px
  - 예: "거북이가 따라오고 있어요"
  
- **Body (본문)**: Sans-serif Regular
  - 크기: 12-13px
  - 예: "3km 러닝"
  
- **Caption (캡션)**: Sans-serif Regular
  - 크기: 10-11px
  - 예: "오후 3:30"

### 텍스트 스타일
```
큰 제목: 18px Bold, #2D5016
제목: 14px SemiBold, #2D5016
본문: 12px Regular, #333
라벨: 11px Regular, #666
캡션: 10px Regular, #999
```

## 🎯 UX 카피 배치

### 5가지 핵심 메시지

#### 1. "오늘의 Hop 완료!" 
**언제**: 일일 인증 후
**어디**: 홈 화면 카드, 완료 팝업
**톤**: 축하하고 긍정적 ✨
```
크기: 14px Bold #7FB069
맥락: "오늘의 Hop 완료! 🎉 3.2km 러닝"
```

#### 2. "거북이가 따라오고 있어요 🐢"
**언제**: 사용자가 거북이에 뒤떨어져 있을 때
**어디**: 알림, 홈 화면 상태 메시지
**톤**: 친근하고 게임 같은 경쟁 느낌
```
크기: 12px Regular #5D8F3F
맥락: "거북이가 따라오고 있어요 🐢 Hop하세요!"
```

#### 3. "토끼가 잠들기 전에 Hop 하세요"
**언제**: 하루를 거의 다 놓쳤을 때 (밤 10시 이후)
**어디**: 알림, 팝업, 푸시 알림
**톤**: 약간의 긴급감 + 따뜻한 격려
```
크기: 12px Regular #8B5A3C
맥락: "토끼가 잠들기 전에 Hop 하세요 💤"
```

#### 4. "3일 연속 Hop 성공!" 
**언제**: 연속 달성 시
**어디**: 배지, 축하 카드, 통계
**톤**: 자랑스럽고 축하하는 분위기
```
크기: 13px Bold #E67E22
배지: 🔥 3일 연속
맥락: "3일 연속 Hop 성공! 계속해봐요!"
```

#### 5. "이번 레이스의 승자는 토끼입니다"
**언제**: 경주 종료 시
**어디**: 결과 화면
**톤**: 축하와 희열
```
크기: 16px Bold #7FB069
맥락: "이번 레이스의 승자는 토끼입니다 🐰"
또는 (50% 미만 시): "이번 레이스의 승자는 거북이입니다 🐢"
또는 (50% 동일 시): "비겼습니다! 🤝"
```

## 🎬 애니메이션 가이드

### 토끼 이동
```
트리거: 인증 완료
지속: 500ms
효과: 
- 토끼가 오른쪽으로 점프하며 이동
- 거리만큼 프로그레스 바 업데이트
- "오늘의 Hop 완료!" 메시지 표시
```

### 거북이 따라오기
```
트리거: 매일 자정 (자동 업데이트)
지속: 1000ms
효과:
- 거북이가 느리게 오른쪽으로 이동
- 알림: "거북이가 따라오고 있어요 🐢"
```

### 토끼 자기
```
트리거: 하루 종일 미인증 시
지속: 정적
효과:
- 토끼 이미지가 누워있는 모습
- 알림: "토끼가 잠들기 전에 Hop 하세요"
```

### 연속 달성 애니메이션
```
트리거: 3일/7일/30일 연속
지속: 1500ms
효과:
- 배지가 흔들리며 나타남 (shake)
- 불꽃 이모지 추가 🔥
- 알림: "3일 연속 Hop 성공!"
```

### 결과 화면 애니메이션
```
트리거: 경주 종료
순서:
1. 🎉 축하 아이콘 bounceIn (300ms)
2. "경주 종료!" 페이드인 (200ms)
3. 우승자 카드 슬라이드업 (500ms)
4. 프로그레스 바 애니메이션 (1000ms)
5. 버튼 페이드인 (300ms)
```
